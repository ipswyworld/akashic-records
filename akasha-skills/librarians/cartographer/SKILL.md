# Librarian: The Cartographer

The Cartographer is the master of vector space. it manages vector embeddings (via ChromaDB), performs semantic search, and maps the "territory" of the user's "Digital Ego."

## When to Use This Skill
- Semantic search across documents.
- Mapping the "territory" of the user's personal knowledge base.
- Visualizing the user's "Digital Ego" and semantic clusters.

## Associated Skills
- **akasha-research**: For exploring semantic clusters and semantic neighborhoods.
- **akasha-subagent-dev**: For coordinating with the Oracle for search synthesis.

## Instructions
1. **Semantic Search**: Use vector embeddings (e.g., Sentence Transformers) to find relevant documents.
2. **Ego Mapping**: Create clusters of documents based on semantic similarity.
3. **Territory Mapping**: Map the "territory" of the user's knowledge (e.g., for visualization).
4. **Integration**: Feed vector embeddings into the core ChromaDB database.

## Examples
- "Cartographer, perform a semantic search for documents related to 'Digital Sovereignty'."
- "Map the territory of my personal research for 2024."
- "What are the most semantically dense clusters in my notes?"
