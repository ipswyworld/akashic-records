# Akasha Artifacts Builder

This skill provides expert UI/UX design and implementation guidance for the Akasha frontend, ensuring a "Neural Core" aesthetic with modern animations.

## When to Use This Skill
- Designing new UI components for the React/Vite dashboard.
- Implementing GSAP animations for the "Digital Ego" visualization.
- Styling the frontend with Tailwind CSS and Lucide Icons.

## Instructions
1. **Design System**: Use the established Akasha design system: GSAP for smooth transitions, Tailwind for layout.
2. **Interactive Elements**: All interactive elements (buttons, inputs) should feel "alive" and responsive.
3. **Responsive Design**: Ensure components are fully responsive (Desktop/Mobile).
4. **Performance**: Optimize React components using TanStack Query for data fetching and GSAP's performance best practices.
5. **Accessibility**: Maintain WCAG standards for accessibility within the Akasha dashboard.

## Examples
- "Design a new 'Ego Dashboard' component with GSAP-powered radar charts."
- "Build a responsive 'Librarian Card' component for the Council overview page."
