# Akasha Mobile Sensor Scaffold
This directory contains the lightweight scaffold for simulated mobile nodes that stream sensor data (GPS, Audio) into the Akasha Universal Library.

## Features
- Simulated GPS coordinate tracking.
- Mock audio data capture.
- Secure transmission to the central Akasha node.

## Usage
Run the `sensor_client.py` to start streaming mock data:
```bash
python sensor_client.py
```
